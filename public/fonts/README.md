# Font Files Required

This directory should contain the DIN Pro font files:

- `DINPro-Regular.woff2`
- `DINPro-Regular.woff`
- `DINPro-Bold.woff2`
- `DINPro-Bold.woff`

These fonts are used in the header component for a consistent brand experience.

If you don't have the DIN Pro font files, the system will fallback to the default sans-serif font.
